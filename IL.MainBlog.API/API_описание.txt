Если добавить `[ValidateAntiForgeryToken]` к методу, к примеру `Login`, то это означает, что запросы на этот метод должны содержать действительный токен предотвращения CSRF. Если запросы
не содержат этот токен, сервер вернет ошибку `400 (Bad Request)`.

Для выполнения метода `Login` в Swagger, необходимо отправить запрос, содержащий действительный токен предотвращения CSRF. Обычно этот токен генерируется и отправляется автоматически при отправке
запросов через форму на веб-странице. Однако при использовании Swagger, вам нужно будет сгенерировать и включить этот токен в запрос вручную.
